`atri`是一个QQbot，她可以：

* 背单词（英日双语）
* 随机数
* 带饭记账

## 环境安装
1. [使用conda](../coding/python.md#创建环境)创建出所需环境（此处环境名为`atri`）：`conda create -n atri python=3.9`
> nonebot2需要在python>=3.8环境下运行
2. 使用pip安装依赖：
```
pip install nb-cli
pip install nonebot-adapter-onebot
```
3. 参考[随笔中的启动脚本](../essay.md#_20220506)
