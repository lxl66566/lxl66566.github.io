# html

单 html 的优点：

- 对于比较简易的小玩意，写一个 html 单文件要比 GUI 应用简单很多。
- 小
- 跨平台
- 快速部署（静态托管）

缺点：

- [跨域访问限制](./nodejs.md#cors-policy)

## 工具

作为 `<script>` 引入的 js 库，好用。

- 界面构建，简化语法
  - [vanjs](https://vanjs.org/)
  - [solidjs](https://www.solidjs.com/)
- markdown 解析
  - [marked.js](https://marked.js.org/)
