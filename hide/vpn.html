# 免费VPN

警告

自带闭源客户端的免费VPN在安全性方面没有保证，不推荐使用。

免费VPN仅应在无其他安全选择的情况下，接触非敏感信息时使用，且有较高信息泄露风险。 随着审查力度变化，不排除肉身被带走的风险。血的教训（当然不是我open in new window

# clash（推荐）

# 简介

clash是开源的、支持Android与PC的客户端。（Android端请前往谷歌商店）

# 下载地址

项目地址open in new window | PC端官方下载地址open in new window | Android端私链open in new window

# 优势

  • 流畅的图形界面
  • 简单易上手

# 简易配置教程 v0.19.16

  1. 在Profiles中的文本框内粘贴订阅链接并下载。选中下载的配置文件。
  2. 进入Proxies，选择rule，并在下方选择一个可用节点。
  3. 进入General，开启System Proxy。

# 一些提示

  • 这里open in new window提供clash的汉化。
  • 在节点界面点击测速图标测试全部节点的连通性
  • 建议设置配置文件自动更新（Profiles->右击配置文件->Settings->Update Interval）
