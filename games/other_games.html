# UNO扩展规则5.0.1

这是一个高度扩展的线下桌游。由于迭代次数较多，具有较高可玩性。你可以自制一套牌进行游玩。

# 基本规则

UNO扩展规则继承自UNO，底层规则自然便是UNO规则。

以下为对规则的严谨性陈述，当打牌过程中遇到纠纷时请以该严谨性规则为准。对于想要快速上手的新玩家，建议 在阅读基本规则与开局规则后，跳转到精简规则进行阅读。

除了数字牌以外的牌均称作功能牌。黑牌指上表中不带有每色x张字样的牌。

我们将 “当前牌”称为last。 你打出的牌若非黑牌，则需与last数字/功能相同或颜色相同。而不论你打出什么牌，都需要满足该牌的打出条件。除改色 +4外的所有黑牌打出时不会直接改变last。非黑牌打出后将会替换last。

你的每个回合可分为三个阶段：

  1. 判定阶段：处理你身上的当前（激活性）效果。
  2. 摸牌阶段：摸一张牌。
  3. 出牌阶段：按照出牌规则出一张牌 （某些功能牌可以让你出更多牌）。

判定阶段是强制性的，而摸牌阶段与出牌阶段你可选择其一，或两个都选，但不可不选。（说人话：没牌可出就摸）

你需要在你的手牌从多于一张变为只剩一张后喊“UNO!”。若被任意玩家指出没喊则需摸两张。你的手牌在脱离肢体接触时才视为失去。

所有功能牌的功能都可被视为效果。

# 效果分类

# 即时性效果

你（或游戏）需要立刻对它作出回应。

# 持续性效果

持续一段时间，有明确的结束条件。

# 激活性效果

你需要在你的判定阶段作出回应。

# AOE效果

对全局施加效果，暂时停止正常游戏，直到效果结束；你无需立刻回应，而应在效果指定到你后回应。

# 自生效效果

为牌本身添加效果。

胜利条件为打完所有手牌、身上不附带任何效果、且不存在被附加效果的可能性。例如：你打出了你最后的手牌（一张+4牌），但在这张+4生效前你仍未胜利，因为它有传递到你身上的可能性。同理，若你最后的手牌是神手、夹子、手榴弹等等，在效果结算完成前，都不可算作胜利。

# 开局规则

游戏建议采用积分制（当然你也可以不用）。每位胜利的玩家获得x分，x为当前场上存活的人数。复活赛的玩家若胜利则同样计分。

初始手牌数可以选用5,7,9等数字（根据玩家数决定）。当手牌数超过初始手牌数*3时，玩家将不允许从牌堆获得手牌。

每个玩家摸初始手牌后需要决定第一出牌人（建议使用拼点、石头剪刀布、上一局最快胜利者直接担任等方式决定）。第一出牌人可以自由指定顺序（顺逆时针），并翻开牌堆的一张非黑牌作为last。然后游戏由第一出牌人开始出牌。

# 常见效果

# 加牌效果

属于激活性效果，拥有两个变量：当前加牌数与最大加牌值。当前加牌数代表若你接受加牌后需要摸牌的数量。最大加牌值代表只有大于等于该值的加牌效果才可继续叠加（说人话：+2不能叠加在+4上面）。当前加牌数可叠加。最大加牌值取叠加过程中最大值（2或4）。

玩家判定阶段选择接受与不接受加牌效果，（不管做何选择都）跳过他的回合。

  • 接受加牌：摸当前加牌数数量的牌。效果结束。
  • 不接受加牌：施加新的加牌效果（效果需不小于当前最大加牌值），将加牌效果叠加后继续施加给下家。

# 转移牌效果

属于激活性效果。效果施加者可为此效果附加0/1/2张手牌（该附加牌可叠加）。然后将此效果与附加牌传给下家。附加牌在传递过程中应背面朝上，在下家未表明接受与否前不应泄露。若下家翻看附加牌则默认为接受效果。

玩家判定阶段选择接受与不接受转移牌效果。

  • 接受转移牌：获得效果上的附加牌，并获得 禁锢效果（持续性效果）：本回合内你无法打出转移牌直到回合结束解除。 转移牌效果结束。
  • 不接受转移牌：打出同类型转移牌，给该效果附加0/1/2张手牌，将效果叠加后施加给下家并跳过自己的回合。过程中不翻看附加牌。

# 眩晕效果

属于激活性效果。

玩家立刻跳过自己的该回合。效果结束。

# 虚弱效果

属于持续性效果。

不可打出pass、反弹、传递直到效果解除。

解除条件：你的判定阶段结束后自动解除。

# 禁

即时性效果：立刻跳转到使用者下两家，开始该玩家的回合。

提示

禁 牌拥有功能牌中最高的优先级。

# 转

即时性效果：逆转出牌顺序。

# +2

激活性效果：对下家施加加牌效果：当前加牌数 = 2 , 最大加牌值 = 2。

# +4

即时性效果：更改last的颜色为指定颜色（数字/功能视为无类型）。

激活性效果：对下家施加加牌效果：当前加牌数 = 4 , 最大加牌值 = 4。

# 改色

即时性效果：更改last的颜色为指定颜色（数字/功能视为无类型）。

# 无限

自生效效果：可以指定为任意数字使用（不改变颜色）。但不可在赌怪（已弃用）的AOE效果中使用。

# 传递

分为<=4与>=5两种，应视作不同类型牌。

打出条件：last同时满足颜色条件与数字条件，或last为同类型牌。

激活性效果：为下家附加转移牌效果。

# 神手

即时性效果：与下家互换手牌。该效果优先级仅次于禁，不可被转移。

# 夹子

即时性效果：抽取下家两张手牌，若不足两张则抽取全部。

# 反弹

即时性效果：先施加转效果再施加pass效果。

# pass

即时性效果：将自身加牌效果与转移牌效果转移给下家。（若无效果则当作白牌打）

# 决斗

分为颜色决斗与点数决斗两种。

即时性效果：指定任意一名玩家进行决斗。胜者弃两张牌（不足则全弃），败者加四张牌，平局算皆败。

  • 颜色决斗规则：
    点击展开

    发起人选择5个颜色（包括黑色）中的一个，接受者选择不同颜色，然后发起人从牌堆翻起一张牌并公示，猜对胜利，猜错失败。（选择颜色的顺序可由发起人决定。）

  • 点数决斗规则：
    点击展开

    两人各从牌堆中翻一张牌拼点，点数大的获胜。

# 天选之子

即时性效果：只保留一张数字牌，弃掉其他所有牌。若无数字牌则全弃。

# 炸弹

即时性效果：以发起者为中心，发起者摸x张牌，两侧玩家视距离依次摸x-1张牌，x-2张牌...x为当前场上存活玩家数。

# 缴械

激活性效果：对下家施加缴械子效果。

缴械子效果：玩家判定阶段选择接受与不接受缴械效果，（不管做何选择都）跳过他的回合。效果结束。

  • 接受缴械：弃两张功能牌。
  • 不接受缴械：摸4张牌。

# 手榴弹

AOE效果：指定四种基础颜色之一并将手榴弹传给下家。接受者需打出该颜色的牌并指定颜色后继续传递给下家。否则，将手牌全弃，然后摸与初始手牌数相等的牌，手榴弹效果结束。过程中打出的牌均计入last。

# 紧箍咒

激活性效果：对任意玩家施加紧箍咒子效果。

紧箍咒子效果：属于持续性效果，拥有变量：有效手牌数。对无效果的玩家而言，有效手牌数与当前手牌数相等。施加效果瞬间，有效手牌数减半（向上取整）。紧箍咒效果可叠加。

玩家在任何情况下不许从牌堆中摸牌（不影响从其他玩家处获得牌）。

解除条件：手牌数不大于有效手牌数。解除后本次有效手牌数作废。

# 复活

即时性效果：可选择对自己用或对已下场玩家用。

  • 对自己用：摸三张牌。
  • 对已下场玩家用（一局仅限一次）：令之重新回到场上，摸一张牌，获得眩晕效果。

# C4

AOE效果：由发起者开始，出一张数字牌使C4被向下传递x家，x为此数字牌的大小。然后跳转到接到C4的玩家的回合，重复该过程。若某位玩家无法打出数字牌，则C4爆炸，使该玩家弃掉所有手牌并重新摸等于初始手牌数的牌，并由发起者的下家开始新的回合。（若被炸的就是发起者的下家，则跳过此玩家）过程中的数字牌将替换last。

# 抬杠

AOE效果：对自己施加抬杠子效果并立刻判定。

抬杠子效果：属于即时性效果，拥有变量：x，初始值为0。

打出不小于x的数字牌并传给下一家，并把数字赋给x。若打不出则效果结束，摸4张牌，由发起者的下家开始新的回合。过程中的数字牌将替换last。

# 奖励

即时性效果：必需打出所有与奖励牌颜色相同的牌，立刻获得眩晕效果与虚弱效果，并可选择是否打出一张任意颜色的神手。（若神手与奖励同色被强制打出，也可选择被算作打出）

  • 若选择打出神手，你可以将刚打出的所有与奖励牌颜色相同的牌盖上，强制让除你以外的每位轮流抽一张。（抽完即停，抽不完需要轮流到抽完为止）

# 已弃用功能牌

此处存放一些曾经构思过但因为各种原因被弃用的功能牌。这些功能牌的解释不具有严谨性。

# 组合牌

单独打出为白牌效果，可以选择同时打出“组合牌上写的数字组合，或者4张同数字炸弹”。

# 天弃之子

出此张牌，必须不断摸牌直至摸出数字牌或者功能牌（摸牌前自选）。

# 跳转

将当前回合（包括当前效果，相当于pass）跳转到指定玩家，不改变顺序与当前牌。

# 赌怪

双方各出4张数字牌（若不够则补至4张），混合后，以0作为初始数字，相互一人取一张按自定义顺序进行加乘减除，最后将结果拿来比大小，赢得可选择自己弃2张或对手摸4张。若出现除以零则按照除以无穷小的正数计算。

# 精简规则

此精简规则默认继承自UNO基础规则。

效果优先级：禁 > 神手 > 夹子 > 炸弹，决斗，缴械，抬杠，c4等一般功能牌 > 反弹，pass，跳转 > 传递，加牌。

非“改变颜色的牌（改色，+4）”不改变当前牌。视为打出的牌会成为当前牌。

玩家胜利当且仅当无手牌且身上无任何效果。C4，抬杠等视为对全局玩家添加效果。

各种功能牌的作用：

# 反弹牌

收到的效果反弹，且反转出牌顺序。若无效果可当作逆转打。

# pass牌

效果传给下一家，若无效果可当作白牌打。

# 决斗牌

任选一个人单挑分别是拼大小和猜颜色，赢的人扔两张牌，输的人加四张牌，平局算皆输。数字决斗为两人各翻一张牌拼点；颜色决斗为发起人选择5个颜色（包括黑色）中的一个，接受者选择不同颜色，然后发起人翻起牌顶一张牌并公示，猜对为胜利。

# 无限牌

可以充当任意数字。不可用于赌怪中。

# 天选之子

只保留一张数字牌，弃掉其他所有牌（若无数字则直接胜利）。

# 传递牌

在上一张牌同时满足颜色条件与数字条件时打出，或在上一张牌为同类型牌时打出。可以任选0-2张牌传递给下家。传递牌可叠加，但传递过程中不可看牌。当有人选择接收传递后，则他的这一阶段不可再打出传递牌（即使同类型）。

# 神手

与下家换牌，具有极高优先级。

# 炸弹

以自己为中心向两侧扩散，自己加全场人数的牌，每扩散一个人加牌数减一。

# 缴械

下家选择：弃两张功能牌，或者加四张牌。并跳过自己的这个回合。

# 夹子

抽下家两张牌，若不足两张则抽取全部。

# 手榴弹

指定一个颜色（仅四种基础颜色）并将手榴弹传给下家。下家需打出该颜色的牌并指定颜色后传给下家。否则，将手牌与手榴弹全弃，然后摸与初始手牌数相等的牌。

# 紧箍咒

对任意玩家使用，该玩家在任何情况不许从牌堆中摸牌（不影响从其他玩家处获得牌），直到手牌数降至一半（向上取整）及以下后，解除紧箍咒效果。紧箍咒效果可叠加。

# 复活

对自己用：自己摸三张牌。对已下场玩家用（一局只能用一次）：重新回到场上并摸一张牌；复活后跳过自己的第一个回合。

# C4

出此牌的人开始，出一张数字牌使c4被向下传递x家，x为此数字牌的大小。然后跳转到接到c4的玩家的回合重复该过程。若无法打出数字牌，则直接爆炸，使拿着c4的玩家弃掉所有手牌并重新摸初始牌，并由出牌者的下家开始新的回合。过程中的数字牌视为打出。

# 抬杠

由出牌者开始出数字牌，并传给下一家，下一家需要打出大于等于上张数字的数字牌，若打不出则抬杠结束，失败者摸4张，由出牌者的下家开始新的回合。过程中的数字牌视为打出。
