                                  # LLM

                                  建议先看看别人做的合集：awesome-free-chatgpt - githubopen in new window

                                  LLM：Large Language Models，大型语言模型。下述均为本人总结，可免费使用。

                                  • BAI Chatopen in new window：GPT-3.5 模型。也可以使用 Telegram 与其进行交流open in new window。
                                  • new bingopen in new window：GPT-4 模型，好处是可以获取实时信息，会给出来源。坏处是被微软大削弱后，不会帮你写作业（
                                    • 需要使用科学上网，若自动重定向到 (cn).bing.com 则需要：检查代理，清除 Cookies，并在 microsoft 账户设置中关闭严格搜索。
                                  • Claudeopen in new window：据说是 OpenAI 独立出来的员工创办的。无法直接使用（需要美国身份），请通过 slack bot 使用open in new window。
                                  • phindopen in new window：搜索编程有关问题。会给出来源。
                                  • wrtn.aiopen in new window：韩国网站，免费的 GPT-4，无需科学上网；支持画图，但新政策不支持中文，需要先翻译再使用。
                                  • Bard：Google 官方提供的 LLM。目前还在 Beta。
