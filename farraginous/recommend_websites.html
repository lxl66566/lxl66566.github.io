AI 工具

LLM

这里的条目已经过时了，现在 LLM 跟雨后春笋一样，太多了。

建议先看看别人做的合集：awesome-free-chatgpt - github

LLM：Large Language Models，大型语言模型。下述均为本人总结，可免费使用。

  • ChatGPT：GPT 官方。此链接直接进入 gpt-4o 模型，一个免费的 GPT-4 水平的 LLM，发布于 2024.05。
  • Gemini（前身 bard）：免费 api 无限用还是爽的
  • Copilot：Win11 有集成；可以到 edge 浏览器里使用。
  • Claude：据说是 OpenAI 独立出来的员工创办的，编码能力很强。无法直接使用（需要美国身份），请通过 slack bot 使用
  • BAI Chat：多模型，有一部分免费。基本上算一个镜像吧。
    • 也可以使用 Telegram 与其进行交流，免费，但是不能更改模型，而且比较笨。
  • perplexity
  • wrtn.ai：韩国网站，免费的 GPT-4，无需科学上网；支持画图，但新政策不支持中文，需要先翻译再使用。
  • 搜索型工具，给出来源
    • phind：搜索编程有关问题
    • devv
  • cohere: do not support Chinese; normal ability, less than GPT-4.
  • deepseek：国内的开源大模型，智商一般，不过网页免费，API 便宜量大
  • 论文助手，翻译：
    • kimi
    • ReadPaper
  • notebooklm：google 家的免费资料总结工具
  • Grok：马斯克的免费 AI，还是比较有诚意的。

公式识别

现在的公式识别已经被 AI 统治了。你当然可以使用一些 VLM + 提示词进行公式识别；不过这里也提供一些其他的特化方式：

  • SimpleTex：免费手写/图片公式识别。模型不开源，而在线 API 比较火爆，可能需要排队。
  • Doc AI：VLM 识别系统，准确率高。

我对传文件的网页需求是 P2P，即无需将文件存储在云端服务器。

  • FilePizza
  • TransferFiles

pastebin

网上分享一些日志 / 报错啥的，不要刷屏，用 pastebin 服务。

很多 pastebin 是自建的，说不上啥时候就寄了。因此如果有长期保存的需求建议还是用些更稳定的。

  • pastebin.com，好记，免登录，以前一直用；但是有内容机审（贴日志的时候可能误报敏感信息）。
  • gist.github.com，github 的服务，可以在日志下方讨论，挺不错。缺点可能是需要登录，不过谁还没有个 github 账号呢？
  • pb：支持 paste from cli，farseerfc 大佬的 pb
  • pastes.dev, termbin：极简风格
  • pb.nichi.co：仅支持 paste from cli，极极简
  • bin.outv.im：注重加密
  • https://netcut.cn/：最长 3 年，有敏感词检测
  • https://pb.moku.llk.moe/：编程语言原神写的，支持语法高亮。

然后还有终端录制。。虽然我不喜欢这样引入不必要的体积膨胀，但还是做个记录。
