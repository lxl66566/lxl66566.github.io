# 读书
***缅怀过去是老了的证明。但我必需直面这~~羞耻~~的过去。***

我自小就爱读书。小学爱看杂志 *（特别是《我们爱科学》）* 和其他故事书，初中时爱看玄幻小说，而高中主要看图书馆/图书角有的书（什么都看）。但是自上大学来，对书的兴趣越来越小了。

此处列举一部分，当作缅怀我的青春了。
## 网络小说
我曾经很爱看网络小说，主要类别有玄幻，爽文，日轻。此处列举出有记录的部分。（大概算是大学&高中部分，更早的基本上都记不住了）

我最喜欢看的是科技相关的超自然类小说，其中首推《走进修仙》。

刚开始看网络小说时我能一天10小时，坚持几天。（当然前提是假期）<span class="heimu" title="你知道的太多了">*疫情网课期间好像有例外诶...*</span> 之后视精彩程度逐渐放缓阅读速度。
