# Rust
## 介绍
Rust是一门系统编程语言，专注于安全，尤其是并发安全，支持函数式和命令式以及泛型等编程范式的多范式语言。Rust在语法上和C++类似，但是设计者想要在保证性能的同时提供更好的内存安全。——[百度百科](https://baike.baidu.com/item/Rust%E8%AF%AD%E8%A8%80)

### 为什么推荐
#### 优点
* 高性能（与python相比）
* 函数多，语法简单（与C++相比）
* 安全，自动释放内存，所有权机制（与C++相比）
* 开发社区激进，更新频繁
#### 缺点
* 限制条件多，难以通过编译
* 使用群体较少，网上缺少不常见问题的解决方案
* 学习曲线陡峭
